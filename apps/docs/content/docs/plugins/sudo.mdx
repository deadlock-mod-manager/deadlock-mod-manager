# Sudo Plugin

The Sudo plugin provides administrative controls and developer features for advanced users.

![Sudo Plugin – Active](/assets/plugins-sudo-1.png)

## Quick Start

1. Go to Settings → Plugins
2. Find "Sudo" plugin
3. Click "Enable"
4. Click "Open" to access administrative controls

## Features

- **Administrative Controls**: Access to advanced system settings
- **Developer Features**: Tools for debugging and development
- **System Override**: Bypass certain restrictions when needed

## Usage

The Sudo plugin provides a settings interface with usage instructions. Enable the plugin to access administrative features that are typically restricted to regular users.

## Important Notes

<Callout type="warning">
**Administrative Access**
- This plugin provides elevated privileges
- Use with caution as it can modify system behavior
- Intended for developers and advanced users only
</Callout>
