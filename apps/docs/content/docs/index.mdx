## Overview

Deadlock Mod Manager is a comprehensive, user-friendly application that simplifies the process of discovering, installing, and managing mods for Valve's Deadlock game. Built with modern web technologies and featuring a sleek desktop interface, it provides a seamless experience for both casual users and mod enthusiasts.

### Key Features

- **üîç Mod Discovery**: Browse and search through an extensive collection of Deadlock mods
- **‚ö° One-Click Installation**: Install mods with a single click - no manual file management required
- **üì± Modern Interface**: Clean, intuitive design built with React and Tailwind CSS
- **üîÑ Automatic Updates**: Keep your mods up-to-date automatically
- **üíæ Download Management**: Track and manage all your mod downloads in one place
- **‚öôÔ∏è Advanced Settings**: Customize installation paths and preferences
- **üåê Multi-language Support**: Available in multiple languages with community translations
- **üñ•Ô∏è Cross-Platform**: Available for Windows and Linux

### Technology Stack

- **Desktop Application**: Built with Tauri v2 for a native, performant desktop experience
- **Frontend**: React with TypeScript for type-safe, maintainable code
- **Styling**: Tailwind CSS v4 for modern, responsive design
- **Backend API**: Bun and Hono for high-performance mod data synchronization
- **Database**: PostgreSQL with Drizzle ORM for robust data management
- **Build System**: Turborepo for efficient monorepo development
