New to Deadlock Mod Manager? Start here:

1. **Install** - [Download and install](/using-mod-manager/installation) the mod manager for your platform
2. **Setup** - [Configure your game path](/using-mod-manager/getting-started) and preferences
3. **Explore** - [Browse and install mods](/using-mod-manager/features) from the community
